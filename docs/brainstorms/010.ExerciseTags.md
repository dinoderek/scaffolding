# Exericse tags
Add the ability to add tags to exercises when logging them 

A tag can be added to a particular exercise log within a session.
Tags are private to the specific exercise.
Tags will be used in the analytics feature as an optional breakdown of the specific exercise.

Examples of tags:
- Record using a particular grip on a lat pulldown machine
- Record using a particular machine at a gym that has multiple copies of the same machine 

## Experience
When in the session recorder or in edit session:
* The user can quickly add tags by typing the tag nam
* When adding tags the use can quickly choose a tag already previous applied *to the exercise* or create a new tag
* When adding tags the user can enter a manage mode where it can delete tags, show previously delete tags, and undelete tags. Deleting tags just hides them from the selector moving forward.
* The user can quickly remove tags from an exercise
